Theme Installation

Installing Doris is easy and quick.

If you had the Shopify site and now want to change with this theme for your site, follow the instruction below:

Note: Extract the purchase package first, use file the doris1-rxx.zip or doris2-rxx to upload to your Shopify site

1. In your Shopify Admin Dashboard go to:

  • Online Store > Themes

  • Scroll to More themes area

  • Click: Upload theme button

  • Click on Choose File button and select doris1-rxx.zip or doris2-rxx file, then click: Upload

2. Shopify will auto upload & Install the Theme. Waiting some seconds and you will see it completed in More themes area

  • Click to Actions link and select Publish

  • Click: Publish

Customize Editor

This is the Editor help you easy to change the setting options of the theme.

In Online Store > Themes area

  • Click on: Customize button

In your Customize Editor will have 2 tab:

  • Sections: show options/settings of different page

  • Theme Settings: all options/settings for all pages.

Color

Go to Theme settings > Color Settings

Click to the color that want change, the Color Picker will appear and easy to pick custom color.




Typography

Go to Theme settings tab, you will see Typography option to import and setup the font

Shop Logo

Go to Sections tab, select Header option, scroll to Logo Shop area. Upload Logo image that you want.





Shop Favicon

Go to Theme Settings tab, select General Settings > Favicon option, upload Favicon image that you want.





Currency

Depend the theme style, will include or not. But all theme styles support Currency. If you want active it, contact with Support staff.

Go to Theme Settings tab, select Currency option.

After active and it don't work, try with the solution here

Translate Editor

In default package, our theme only include English version. But our theme support Translate Edior of Shopify. It help translate the site to any language.

This function also help you change/edit the text to suitable with your idea.

Go to Admin Shopify Dashboard > Online Store > Themes > Actions button > Edit Languages

Select group tab and change the text that you want
